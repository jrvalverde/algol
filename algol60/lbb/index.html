Identifiers

Arbitrary sequences of letters and digits.
The first symbol must be a letter.
 
 
Variables
Simple variableName
Subindexed variableName[i, j]

 

Numbers

Integer 12345
Real with decimal point '.' or exponent '10'

There must be at least one digit after the decimal point.
For very big or tiny real numbers an exponent with base 10 is used. The exponent must be entireand purely numeric.
Every variable type must be expressed before using it.
Types may be mixed in a single arithmetic expression.
Examples:
    +35    35    -35
    +3.5    .35    -35.0
    3.510-3    -.35105
 
 

Booleans

May have either of two values, true or false.
 

Standard functions

abs(u) absolute value of u: |u|
sign(u) +1 if u > 0
0 if u = 0
-1 if u < 0
entier(u) the biggest integer  u
sqrt(u) square root of u
exp(u) exponential of u (e raised to u)
ln(u) natural logarithm of u
sin(u) sine of u (radians)
cos(u) cosine of u (radians)
arctan(u) arctangent of u (u between -PI/2 and +PI/2)

Reals are automatically rounded as rounded(x) = entier(x + 0.5)
 

Operators

1 Exponentiation
2 multiplication, division, integer division ×, /, ÷
3 addition, substraction +, -

In integer division, the quotient is rounded to the integer immediately below.
Expressions are evaluated from left to right according to parentheses ()
a / b is ALWAYS real
a ÷ b requires a and b to be integers

Arithmetic expressions may contain numbers, variables and functions.
Variables must have received a value prior to being used.
 

Assignment statement

The rule to assign a value to a variable is
 
left hand := right hand
simple variable
indexed variable		 := arithmetic expression
conditional expression
boolean expression
assignment

In case of multiple assignment, all variables in the left hand side must be of the same type.
Subindexes in all indexed variables in the left hand are computed with the current values from left to right.
The right hand side is evaluated.
Its value is assigned to variables in the left side.

Compound statements

State that certain statements must be considered jointly. Statement parentheses are used to enclose the statements:
begin ... end;
May be nested.
Statements preceding an end don't need an ending ;
